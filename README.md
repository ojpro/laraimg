# imgs
## Overview
Local Images Gallery, where you can store,edit, and delete images. plus many functionality like short links and copylink to clipboard 
## Features
1. CRUD Images
2. Short URLs
3. Copy URL to clipboard
4. Unsplash Like Gallery
5. Responsive layout
6. Search for images
7. Slider
8. Coming soon.

## Used Tools
- Larevel
- vuejs
- vitejs
- Tailwindcss
- Clipboardjs
- Sweatalert2
